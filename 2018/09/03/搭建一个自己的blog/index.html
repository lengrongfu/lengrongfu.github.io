搭建一个自己的blog

Posted on 2018-09-03 11:18:34 Edited on 2018-09-03 22:47:34 In Blog Views:
Symbols count in article: 1.1k Reading time ≈ 1 mins.

要求

  • Git账号和基本的Git命令使用。
  • 会用Markdown文章。
  • 会配置基于SSH的Git访问。

环境

使用 GitHub Pages 和 hexo 搭建一个属于自己的无限流量的博客系统。自己不需要拥有服务器，不需要拥有域名。

使用Mac做教程，其他系统请自行研究。

搭建

  • 第一步：创建一个仓库
  • 第二步：拉取到本地仓库
  • 第三步：写index.html
  • 第四步：提交

之后在浏览器中输入https://username.github.io. 就可以进行访问了。可直接参考官网步骤。

  • 第五步：安装hexo

  • 第六步：初始化项目

  • 第七步：部署项目到GitHub Pages上

主题设置

我使用的是 next 主题，如果需要使用其他的，请官网找，设置方法一样。

图片插入问题

使用 hexo-asset-image 来使用本地的图片，不使用其他存储。具体使用方法参考官方配置。

其他问题参考

  • 多个ssh问题,参考：http://riny.net/2014/git-ssh-key/
